# klyx-terminal/homebrew-klyx

Homebrew tap for [Klyx](https://github.com/klyx-terminal/klyx).

## Install

```
brew tap klyx-terminal/klyx
brew install --cask klyx
```

After install, on first launch macOS may quarantine the unsigned bundle.
Right-click `Klyx.app` in Applications and choose **Open**, or run:

```
xattr -d com.apple.quarantine /Applications/Klyx.app
```
